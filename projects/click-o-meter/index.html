Тестовый проект «Click-O-Meter»

2021

Репозиторий

Проект «Click-O-Meter» это тестовый проект на Laravel 5 по созданию простой системы веб-аналитики с картами кликов.

Реализация состоит из 2ух компонентов:

  • Js-скрипт, установленный на сторонние сайты, отслеживает клики пользователей: время и координаты.
  • Сайт-админка, на котором можно увидеть информацию для каждого мониторируемого сайта: карта кликов (подобно Яндекс метрике) и график распределения активности пользователя по времени суток.

Скрипт встроенный на стороннем сайте отправляет данные на серверную часть проекта по API. Принимающая сторона агрегирует данные в базу данных. Реализация проекта полностью на свое усмотрение.

Используемые технологии

  • PHP 7.1
  • MySQL 5.7
  • Git
  • Composer
  • Npm

Фреймворк: Laravel 5.8

Библиотеки:

  • Bootstrap - основной шаблон и тестовые страницы
  • Chart.js - построение графиков на Canvas
  • Simpleheat - построения тепловых карт на Canvas
Примечание:

В связи с тем, что проект является тестовом, функционал реализован в минимальном объеме. Визуализация данных выполняется по первому сайту который будет добавлен в БД.

Скриншоты:

  • График распределения кликов по времени за последние сутки
  • График распределения кликов по времени за последние сутки по одной выбранной ссылке
  • Тепловая карта распределения кликов
  • Тепловая карта распределения кликов, с максимальным радиусом отображения клика
  • Тепловая карта распределения кликов, с максимальным радиусом и размытием при отображении клика
