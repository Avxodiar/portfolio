Учебный проект по изученнию Vue.js

2021

Репозиторий Реализация

Проект содержит в себе 2 "подпроекта", реализованных с использованием компонентов и маршрутизации, отображаемых на отдельных страницах.

  • Список дел (Todo List)
  • Игра "Крестики-Нолики"

Приложение "Список дел" позволяет добавлять и удалять задачи, менять порядок их расположения в списке, имеется возможность отмечать выполенные дела, а кроме того имеется поиск осуществляющий фильтрацию списка дел по совпадению введенной строки. Данные хранятся в браузерном хранилище LocalStorage

Приложение "Крестики-Нолики" это логическая игра между двумя противниками на квадратном поле 3 на 3 клетки. Пользователь играет «крестиками», а компьютер — «ноликами». Игроки по очереди ставят на свободные клетки поля 3х3 знаки (один всегда крестики, другой всегда нолики). Первый, выстроивший в ряд 3 своих фигуры по вертикали, горизонтали или диагонали, выигрывает.

Используемые технологии

  • Vue.js, Vue Cli
  • Npm
  • Git
  • Html, Css, фиксированная верстка

Скриншоты:

  • Список дел. Фильтрация по статусу
