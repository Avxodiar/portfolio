Учебный проект «Todo List»

2021

Репозиторий Реализация

Проект «Todo List» это учебный проект на Vue.js по созданию одностраничного сайта "Список дел".

Список дел хранится в браузерном хранилище LocalStorage, позволяет добавлять и удалять задачи, менять порядок их расположения в списке, имеется возможность отмечать выполенные дела, а кроме того имеется поиск осуществляющий фильтрацию списка дел по совпадению введенной строки.

Используемые технологии

  • Vue.js, Vue Cli
  • Npm
  • Git
  • Html, Css, фиксированная верстка
