Проект «Наша Дача»

2008

Реализация

«Наша Дача» – Справочник по ягодам.Имеется раздел со списком адресов садово-огородных магазинов.

Ежемесечно публикуется календарь работ садовода и огородника.

А для тех, кто привык сверятся с народными приметам - отображаются актуальные приметы на ближайшие дни.

Используемые технологии

  • HTML
  • CSS
  • PHP 5
  • процедурный стиль c паттерном "Page Controller"

отсутсвует
